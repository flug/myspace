# i3 & Polybar Setup on Ubuntu/Debian & Arch

## English:

### Prerequisites:

- **OS**: Ubuntu/Debian or Arch Linux
- **Desktop Environment**: i3wm
- **Status Bar**: Polybar
- **Terminal**: Alacritty
- **File Explorer**: Ranger

### Installation:

#### On Ubuntu/Debian:

```bash
sudo apt install i3 i3status alacritty ranger
```

#### On Arch:

```bash
sudo pacman -S i3 i3status alacritty ranger
yay -S polybar
```
